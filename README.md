# Cybersecurity Risk Prep Roadmap & Guide

A concise, interactive single-page application designed to help aspiring cybersecurity professionals prepare for interviews focused on Governance, Risk, and Compliance (GRC) and Risk Management.



---

## ‚ú® Features

-   **Interactive Learning Roadmap:** Track your study progress with a dynamic progress bar and checklists. Your progress is saved locally in your browser via `localStorage`.
-   **Four-Phase Structure:** The roadmap is broken down into four logical phases:
    1.  **Foundations:** Core concepts like the CIA Triad and Risk Equation.
    2.  **Frameworks:** Key industry standards like NIST CSF and ISO 27001.
    3.  **Assessment:** Practical skills like Threat Modeling and TPRM.
    4.  **Operations:** Day-to-day concepts like Risk Registers and BCP/DR.
-   **Model Interview Answers:** A curated Q&A section that goes beyond simple definitions, explaining common "traps" and how to frame answers from a business-first perspective.
-   **Buzzword Dictionary:** A quick-reference vocabulary list to help you speak the language of risk management fluently.
-   **Fully Responsive:** Clean, modern UI built with Tailwind CSS that works seamlessly on desktop and mobile devices.
-   **Zero Dependencies:** Built with vanilla HTML, JS, and Tailwind CSS (via CDN). No complex setup or build step is required.
